# Bilgisayar Bakım ve Yazılım Denetimi

Bu repository, bilgisayar bakım işlemleri ve yazılım kurulumu/denetimi hakkında hazırlanmıştır.

## İçerik
- Fiziksel bilgisayar temizliği
- Yazılımsal bakım işlemleri
- Sistem performansı kontrolü
- Görev Yöneticisi kullanımı

## Amaç
Bilgisayarın performansını artırmak ve daha verimli kullanılmasını sağlamak.
